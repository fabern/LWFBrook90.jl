About LWFBrook90.jl

LWFBrook90.jl implements a 1D soil-vegetation-atmosphere transfer model. Intended use cases of this impelemntation are:

  • support of stable isotopes (δ¹⁸O and δ²H)
  • efficient model calibration for data analysis
  • increased model flexibility

To read about the model structure, see section SVAT Model. For a quick start refer to the step-by-step guide in section Example For further details read through the section User Guide or refer to sections Code Listing and Function Documentations for further technical intricacies e.g. for development of the package.

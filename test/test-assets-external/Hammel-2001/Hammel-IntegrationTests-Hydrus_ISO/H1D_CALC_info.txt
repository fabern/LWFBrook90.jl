#==========================================
All binaries within this directory should be able to run with the same set of Hydrus 1D input files.
Details to the specific binaries can be found below.
#==========================================

H1D_CALC_iso.EXE (renamed from H1D_calc.exe)
----------------------------------------------
Description:
- "Isotope enabled" modification of the standard Hydrus1D computational module
- no enrichment of solute (-> isotope) during evaporation
- only applicable, when evaporative fractionation can be neglected
- do not use with the actual concentrations of 18O or 2H, instead use offsets for the delta values to avoid negative values
----------------------------------------------

Date of retrieval: 2020-02-28
Download source link:
https://www.pc-progress.com/Downloads/Public_Lib_H1D/H1D_Calc_Isotope.zip

References:
Stumpp, C., W. Stichler, M. Kandolf, and J. Šimůnek
Effects of land cover and fertilization method on water flow and solute transport in five lysimeters: A long-term study using stable water isotopes
Vadose Zone Journal, 11(1), doi:10.2136/vzj2011.0075, 14 pp., 2012.

#===========================================

H1D_CALC_orig.EXE (renamed from H1D_CALC.EXE)
---------------------------------------------
Description:
- standard version of the Hydrus1D computational module
- enrichment of solute (-> isotope) during evaporation
- only applicable, when there is no evaporation at all
- do not use with the actual concentrations of 18O or 2H, instead use offsets for the delta values to avoid negative values
----------------------------------------------
Date of retrieval: 2020-02-26
Download source link:
